# 智能成绩统计工具

这个工具用于把课程平台导出的 `SA`、`LA`、`TL` 成绩文件自动汇总，并把讨论和慕课成绩填回 `CL_*.xls` 平时成绩登记表。它会自动识别名单、成绩列、满分和目标分制，生成一个可追溯的统计留存表。

## 适用文件

把以下文件放在同一个文件夹：

- `CL_1.xls`、`CL_2.xls` 等平时成绩登记表，或其他 `CL*.xls/xlsx` 名单文件
- `SA_1.xls`、`SA_2.xls` 等作业成绩文件
- `LA_1.xls`、`LA_2.xls` 等互评/实验类成绩文件
- `TL.xlsx` 讨论成绩文件

脚本也会读取文件名包含 `平时成绩登记表` 的登记表作为名单来源。

## 安装

建议使用 Python 3.10 或更新版本。

```bash
pip install -r requirements.txt
```

## 常用命令

只生成统计留存表：

```bash
python smart_grade_collector.py
```

生成统计留存表，并直接覆盖 `CL_*.xls` 的讨论和慕课列：

```bash
python smart_grade_collector.py --overwrite-cl
```

生成统计留存表，并输出 `CL_1_已填.xls` 这样的副本：

```bash
python smart_grade_collector.py --fill-cl
```

指定数据目录和输出文件：

```bash
python smart_grade_collector.py --data-dir ./data --output ./data/成绩统计留存表.xlsx
```

## 计算规则

- SA：优先取 `得分/xx分` 这样的总分列，排除 `第1次得分`、`第2次得分`
- LA：取 `得分` 列，排除互评列
- TL：取 `讨论/` 或 `主题讨论` 列，排除平台总成绩、原始分、测验、作业等列
- 慕课成绩：所有 SA 和 LA 原始分合计后，按总满分换算到登记表中的慕课目标分制，默认 20 分制
- 讨论成绩：TL 讨论原始分按满分换算到登记表中的讨论目标分制，默认 10 分制
- 缺失成绩：按 0 分处理，同时在留存表中记录缺失来源
- 写入 CL：讨论和慕课都会按“四舍五入”写成整数，并尽量保留原表格边框样式

## 自动满分识别

满分按以下顺序判断：

1. 从列名读取，例如 `得分/50.0分`
2. 从登记表说明读取，例如 `讨论10分制`、`慕课20分制`
3. 根据成绩分布推断，匹配常见满分 `5, 10, 20, 30, 50, 100`

每个成绩源的满分和识别来源都会写入 `成绩统计留存表.xlsx` 的 `满分识别` 工作表。如果脚本不能可靠判断，会标记为 `需确认`。

## 输出说明

运行后会生成：

- `成绩统计留存表.xlsx`
- 可选：覆盖或生成填好的 `CL_*.xls`

`成绩统计留存表.xlsx` 包含：

- `成绩明细`：每名学生的 SA/LA/TL 原始分、满分、总分和换算分
- `满分识别`：每个成绩文件的成绩列、满分、识别来源和状态
- `名单来源`：每个名单文件识别出的学生数量和表头位置
- `缺失与重复`：缺失成绩、重复名单、成绩源中有但名单中没有的学生

`成绩明细` 中的汇总列是 Excel 公式。打开文件后，如果手动修改原始分、满分或目标分，慕课和讨论换算分会自动重新计算。
